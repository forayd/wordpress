- Installer Wamp : https://www.wampserver.com/#wampserver-64-bits-php-5-6-25-php-7 (direct dl)
- Installer VsCode : https://code.visualstudio.com/download# (direct dl)

Installation Wamp --> 
  - Définir Chrome comme navigateur par défaut qu'utilise wamp
  - Définir VsCode comme éditeur de texte par défaut
  

Installer le projet en local (sur votre PC) --> 
  - git clone le repository dans le dossier www de wamp
 

- Lancer wamp
- Se rendre sur chrome et taper dans le barre de recherche "localhost/"
- cliquer sur phpmyadmin
- nom d'utilisateur : root 
- pas de mot de passe

- Importer la base de donnée du dossier qui a été cloner "projetgroupeteam7.zip" --> 
  - Crée une nouvelle base de donnée ->
    - Nom de la base (ce que vous voulez car on va importer l'autre base et tout sera remplacer)
    - Pour le codage c'est : utf8_unicode_ci 
    - Créer 
  - CLiquer directement importer 
  - choisir un fichier 
  - ficher a selectionner: projetgroupeteam7.zip
  - cliquer sur Exécuter 
 
 
- Accéder au site --> 
  - sur chrome taper "localhost/projetgroupe"
